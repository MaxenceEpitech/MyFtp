# MyFtp

### Use of Server :
```
$ make re
$ ./server [ip] [port]
```

### Use of Client (example with ftp) :
```
$ ftp [ip] [port]
Username : $ anonymous
Password : $
```

### Project Informations :

MyFtp is a ftp server following RFC959 Protocol.

It had been coded following Epitech's standart.

More informations in MyFtp.pdf.
